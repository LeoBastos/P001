# Projeto Web Oficina
Sistema web para gerenciamento de uma oficina
- em andamento

## Como rodar o projeto?
- Clone esse repositório.
- Crie um virtualenv com Python 3.
- Ative o virtualenv.
- Instale as dependências(requeriments.txt).
- Rode as migrações.
